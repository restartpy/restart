# Examples

Some example APIs based on `RESTArt`.


## Dependencies

1. Install `RESTArt`

    ```
    $ pip install Python-RESTArt
    ```

2. Download examples

    ```
    $ cd /tmp
    $ git clone https://github.com/RussellLuo/restart.git
    $ cd restart/examples
    ```
